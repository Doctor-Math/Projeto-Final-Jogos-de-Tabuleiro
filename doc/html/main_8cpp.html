Função principal do sistema de jogos de tabuleiro.

O programa permite o cadastro, remoção e listagem de jogadores, além de executar partidas de jogos como Reversi, Lig4 e Jogo da Velha. O usuário interage através de comandos específicos.

Comandos disponíveis:

  • CJ: Cadastrar jogador
  • RJ: Remover jogador
  • LJ: Listar jogadores
  • EP: Executar partida
    • R: Reversi
    • L: Lig4
    • V: Jogo da Velha
  • FS: Finalizar sistema

O programa exive um menu com comandos e solicita entradas do usuário para executar as opereações desejadas

Retorna
int Retorna 0 ao finalizar o programa com sucesso.

Exibe o menu inicial do sistema.

Loop principal para processar comandos do usuário.

O loop continua até que o comando "FS" (Finalizar Sistema) seja recebido. Em cada iteração, o programa:

  • Lê o comando inserido pelo usuário.
  • Executa a operação correspondente (como cadastrar, remover ou listar jogadores, ou iniciar uma partida).
  • Exibe uma mensagem de erro para comandos inválidos.

Comando para cadastrar um novo jogador.

Chama a função cadastrarJogador() do módulo Cadastro.

Comando para remover um jogador existente.

Chama a função removerJogador() do módulo Cadastro.

Comando para listar os jogadores cadastrados.

Chama a função listarJogadores() do módulo Cadastro.

Comando para executar uma partida entre dois jogadores.

Solicita o tipo de jogo e os apelidos dos dois jogadores. Valida se ambos os jogadores estão cadastrados antes de iniciar a partida.

apelidoJ1Apelido do jogador 1.
apelidoJ2Apelido do jogador 2.
Exceções
Exibemensagem de erro caso um ou ambos os jogadores não estejam cadastrados.

Verifica o cadastro dos jogadores e executa a partida, se possível.

A função verificarCadastro é chamada para verificar se os jogadores fornecidos (apelidoJ1 e apelidoJ2) estão cadastrados no sistema.
