◆ cadastrarJogador()

void cadastrarJogador ( )

Função para cadastrar um novo jogador no arquivo "Jogadores.txt".

Lê o nome e apelido do jogador e verifica se o jogador já existe no arquivo. Caso o jogador não exista, ele é adicionado ao final do arquivo com estatísticas zeradas.

Observação
O arquivo "Jogadores.txt" deve existir previamente.

void listarJogadores ( )

Função para listar todos os jogadores cadastrados no arquivo "Jogadores.txt".

Imprime o conteúdo do arquivo na saída padrão. Se o arquivo estiver vazio, uma mensagem será exibida informando que não há jogadores cadastrados.

Observação
O arquivo "Jogadores.txt" deve existir previamente.
