Função principal do sistema de jogos de tabuleiro. 

O programa permite o cadastro, remoção e listagem de jogadores, além de executar partidas de jogos como Reversi, Lig4 e Jogo da Velha. O usuário interage através de comandos específicos.

Comandos disponíveis\+:
\begin{DoxyItemize}
\item CJ\+: Cadastrar jogador
\item RJ\+: Remover jogador
\item LJ\+: Listar jogadores
\item EP\+: Executar partida
\begin{DoxyItemize}
\item R\+: Reversi
\item L\+: Lig4
\item V\+: Jogo da Velha
\end{DoxyItemize}
\item FS\+: Finalizar sistema
\end{DoxyItemize}

O programa exive um menu com comandos e solicita entradas do usuário para executar as opereações desejadas

\begin{DoxyReturn}{Retorna}
int Retorna 0 ao finalizar o programa com sucesso. 
\end{DoxyReturn}
Exibe o menu inicial do sistema.

Loop principal para processar comandos do usuário.

O loop continua até que o comando \char`\"{}\+F\+S\char`\"{} (Finalizar Sistema) seja recebido. Em cada iteração, o programa\+:
\begin{DoxyItemize}
\item Lê o comando inserido pelo usuário.
\item Executa a operação correspondente (como cadastrar, remover ou listar jogadores, ou iniciar uma partida).
\item Exibe uma mensagem de erro para comandos inválidos.
\end{DoxyItemize}

Comando para cadastrar um novo jogador.

Chama a função \hyperlink{Cadastro_8cpp_ab63dd84afabba6c1ad653161142bc769}{cadastrar\+Jogador()} do módulo Cadastro.

Comando para remover um jogador existente.

Chama a função \hyperlink{Cadastro_8cpp_af751b98f97728c80ce9b3cc8789d8780}{remover\+Jogador()} do módulo Cadastro.

Comando para listar os jogadores cadastrados.

Chama a função \hyperlink{Cadastro_8cpp_ac0f0834a2a4a602263e22f1fde0e3553}{listar\+Jogadores()} do módulo Cadastro.

Comando para executar uma partida entre dois jogadores.

Solicita o tipo de jogo e os apelidos dos dois jogadores. Valida se ambos os jogadores estão cadastrados antes de iniciar a partida.

Verifica o cadastro dos jogadores e executa a partida, se possível.

A função verificar\+Cadastro é chamada para verificar se os jogadores fornecidos (apelido\+J1 e apelido\+J2) estão cadastrados no sistema. Caso estejam, a função executar\+Partida é chamada para iniciar o jogo entre os dois jogadores, com o tipo de jogo selecionado. Caso contrário, uma mensagem de erro é exibida.
